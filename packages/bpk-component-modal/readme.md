# bpk-component-modal

> Backpack modal component.

## Installation

```sh
npm install bpk-component-modal --save
```

## Usage

```js
import React from 'react'
import BpkModal from 'bpk-component-modal'

export default MyComponent = () => (
  <BpkModal />
)
```
