Living Styles is an up-to-date library of styles for use on Skyscanner's web products.

Installation

Install the module using the git url like so:

npm install git+http://git.prod.skyscanner.local/design/living-styles.git#v0.2.1 --save-dev

For more installation instructions please see the project in Git Lab.
